# CiliCili

一个免费开源的第三方哔哩哔哩电视客户端，专为大屏横屏设计。

## ✨ 特性

- 📺 **电视优化**：专为 Android TV / 横屏大屏设计，左侧导航 + 右侧内容的经典大屏布局
- 🎨 **双风格设计**：Google Material 3 + Apple iOS 两套主题，随时切换
- 🔥 **内容丰富**：首页推荐、热门视频、排行榜、全分区浏览
- 🔍 **搜索功能**：支持视频搜索、热门搜索词
- 📱 **横屏适配**：强制横屏布局，最佳大屏体验
- 😆 **搞怪报错**：告别干巴巴的错误信息，都是骚话
- 💾 **轻量体积**：优化 APK 大小，装起来不占空间

## 📦 下载

从 [Releases](https://github.com/cheymin/CiliCili/releases) 页面下载最新版本。

## 🚀 功能模块

| 模块 | 说明 |
|------|------|
| 首页 | 推荐、热门、排行榜三个 Tab |
| 分区 | B站全分区视频浏览 |
| 搜索 | 关键词搜索视频、热搜榜单 |
| 视频详情 | 视频信息、UP主、相关推荐 |
| 关于 | 作者信息、项目地址、版本号 |

## 🛠 构建

```bash
flutter pub get
flutter build apk --release
```
